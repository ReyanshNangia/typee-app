# Changelog

All notable changes to Typee are documented here. This project follows
[Semantic Versioning](https://semver.org).

## [1.0.0] — 2026-06-27

The first public release. 🎉

### Highlights
- **Floating notepad** that appears anywhere with a double-tap of a modifier key
  (Control, Option, or Command — your choice).
- **Multi-note carousel** — swipe or use `⌘[` / `⌘]` to move between notes,
  drag the page dots to reorder.
- **Rich text** — bold (`⌘B`), underline (`⌘U`), and per-selection font sizing
  with `⌘=` / `⌘−`.
- **Smart lists** — type `1.`, `a.`, or `i.` followed by a space to start an
  auto-continuing numbered, alphabetic, or roman list.
- **Color themes** — tint any note from a warm palette; the scrollbar matches.
- **Live word & character count** in the corner of the window.
- **Appearance controls** — global font-size and window-opacity sliders in
  Settings.
- **Reliable global hotkey** built on a `CGEventTap` that survives sleep/wake.
- **Persistent Accessibility permission** when run as the installed app bundle.
- **Custom notes folder** — keep your notes in iCloud, Dropbox, or anywhere.
- **In-app update checks** with a menu-bar badge when a new version ships.
