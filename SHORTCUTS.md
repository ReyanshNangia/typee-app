# Typee Keyboard Shortcuts

## Window

| Shortcut | Action |
|----------|--------|
| Double Control | Show / Hide Typee |
| Escape | Hide Typee |

## Notes

| Shortcut | Action |
|----------|--------|
| Cmd+N | New note |
| Two-finger swipe left/right | Navigate between notes |
| Overscroll right at last note | Create new note (elastic spring) |

## Text Editing

| Shortcut | Action |
|----------|--------|
| Cmd+A | Select all text in current note |
| Cmd+B | Toggle bold on selection |
| Cmd+U | Toggle underline on selection |
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Cmd+X | Cut |
| Cmd+C | Copy |
| Cmd+V | Paste |
